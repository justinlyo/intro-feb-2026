# Networking




### TCP
- Reliable, two-way communication channel that ensures data is sent and received in the correct order.
- No message loss or duplicates
- One of the many uses is in web browsing (HTTP/HTTPS)




### OSI Model
- Application 
    - Interface to provide network (HTTP)
- Presentation
    - Data is usable (encryption)
- Session
    - Session between devices (establishing/maintaining/ending connections)
- Transport
    - Transfers data reliably (TCP/UDP protocols)
- Network 
    - Routes data between devices across networks
- Data Link
    - Error detection and framing for transmitting physically (ethernet)
Physical
    - Raw transmit over physical media (cables, radio waves)
